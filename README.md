# Pet Nook Tools

Static GitHub Pages framework for a cozy pet tool hub.

## Files

- `index.html` - page structure, SEO shell, content sections
- `guides.html` - launch article library page
- `legal.html` - affiliate disclosure and veterinary disclaimer
- `styles.css` - light/dark visual system and pet animations
- `script.js` - theme toggle and 12 static pet tools
- `ART_DIRECTION.md` - illustration direction and asset prompt library
- `FACT_CHECK.md` - source-backed notes and safety boundaries for tool logic
- `LAUNCH_PLAN.md` - remaining launch tasks
- `robots.txt` - crawler allow rule
- `sitemap.template.xml` - sitemap template that needs the final domain

## Local preview

Open `index.html` directly, or run:

```powershell
python -m http.server 4173
```

Then visit `http://localhost:4173`.
